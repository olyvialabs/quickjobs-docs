# Cambiar contraseña

Forget your password or having trouble logging in? Time for a reset.

## Escritorio
Accede a <span style={{color : '#f97316'}}>**Ajustes**</span>.

<p align="center">
  <img src="/img/change-password/configuracion-desktop-es.png" width="300" />
</p>

Selecciona la opcion <span style={{color : '#f97316'}}>**Cambiar contraseña**</span>.

<p align="center">
  <img src="/img/change-password/change-password-desktop-es.png" width="65%" />
</p>

Escoge una nueva contraseña, despues llena los campos con tu nueva contraseña y guarda los cambios con el boton  <span style={{color : '#f97316'}}>**Guardar**</span>.

<p align="center">
  <img src="/img/change-password/save-password-desktop-es.png" width="65%" />
</p>
